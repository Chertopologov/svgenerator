# Resume Generator — простая версия

Локальный одностраничный генератор резюме: выбор шаблона, фото, динамические секции (образование, опыт), экспорт в PDF и .doc.

Как запустить

1. Откройте файл `index.html` в браузере (двойной клик или через встроенный сервер).
2. Заполните поля, нажмите «Показать/Обновить превью». Выберите шаблон сверху.
3. Для скачивания: нажмите «Скачать PDF» или «Скачать .doc».

Технологии

- Чистый HTML/CSS/JavaScript
- Клиентский экспорт PDF через `html2pdf.js` (CDN)

Замечания и улучшения

- Можно добавить больше шаблонов, печатные стили и настройку полей шрифтов/цветов.
- Для экспорта в настоящий `.docx` можно интегрировать библиотеку `docx` (требует сборки или CDN).
