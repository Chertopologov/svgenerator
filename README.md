# Генератор резюме

Одностраничный веб-генератор резюме с заполнением формы, выбором шаблона, фото, динамическими разделами и экспортом в PDF и DOCX.

## Технологии

- HTML / CSS / JavaScript (без сборки)
- `html2pdf.js` — экспорт в PDF
- `docx` + `FileSaver.js` — экспорт в DOCX
- `flatpickr` — выбор дат
